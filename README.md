# google-photos-tools
Tampermonkey Userscript for Google Photos:
- Delete on key press
- Preloading:
  - Precache next X photos when opening a photo in full screen view
  - Auto-Scroll to precache hundreds high-res photos
  - Grid Warmer: Automatically scrolls down and pre-loads thumbnails in the gallery view to ensure smooth scrolling
